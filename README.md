#  E-Pharmacy Price Comparison App

This project is a simple web application built using Python and Streamlit that allows users to search for medicines and compare their prices from different online sources in real time. It uses an API to fetch data, processes it using Pandas by cleaning and sorting price values, and displays the results in a user-friendly interface along with visualizations like bar charts and pie charts. The main objective of this project is to demonstrate API integration, data cleaning, and basic data visualization while building an interactive application.

Features
- Search medicine name
- Compare prices from different sources
- Find cheapest option
- View price chart

Tech Used
- Python
- Pandas
- Requests
- Streamlit
- Matplotlib
